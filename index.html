História dos computadores

Podemos dividir os computadores em gerações, da seguinte forma:

  • Primeira Geração: Válvulas- Mark-I, Colossus, ENIAC.
  • Segunda Geração: Transistores- De 1959 a 1965.
  • Terceira Geração: Circuitos Integrados.
  • Quarta Geração: PCs - 1971 até hoje - Personal Computers - IBM PC
  • Quinta Geração: Alguns entendem que engloba IA, Computação Quântica, Criptomoedas

O que são Clients?

Clients são os dispositivos que fazem o acesso via Internet, usando protocolo HTTP, de recursos armazenados em servidores.
Por exemplo, posso citar smartphones, notebook, desktops que acessam um determinado site hospedado em um servidor, que por sua vez retorna os arquivos para o client. (HTML, CSS, JS, imagens, áudio, etc).
É no lado do Client (cliente) que funciona o Front End, ou a parte com a qual o usuário interage diretamente. é composto de HTML, CSS, JavaScript como pilares fundamentais.

O que são Servers?

Em uma arquitetura Client - Server podemos entender o servidor com um equipamento (servidor) ou sistema que fornece recursos (ou serviços) para o Client de acordo com as requisições realizadas.
Um exemplo pode ser o processamento feito no lado do servidor para efetuar uma transferência bancária solicitada pelo Client em uma aplicação web. A página acessada pelo cliente envia os dados para o servidor (back end) que processa a operação e depois devolve uma confirmação para o client, na sua tela . (Front End)

Como é uma estrutura básica de um HTML?

Posso citar uma lista de tags que podemos usar para definir uma estrutura básica de um HTML:

  1. html: Define o início e fim do conteúdo HTML.
  2. head: utilizado para definir, por exemplo o título que aparece na aba do browser.
  3. h1,h2,h3,h4,h5,h6: Usados para sub-títulos. H1 é a maior e H6 a menor.
  4. body: Contém toda a parte do HTML que será visível ao carregar a página.
  5. p: TRata-se do parágrafo. Escrevemos textos mais longos nesta tag.
  6. img: Tag usada para apresentar uma imagem, podendo ela estar local ou na Internet.
  7. a: Anchor . Usada para acesar links externos (Internet) ou posições dentro da própria página.
  8. strong: Coloca em negrito e também coloca ênfase em leitores de páginas com áudio.
  9. i: Usado para colocar em itálico.
  10. ol: Define uma lista como ordenada. (numerada).
  11. ul: Define uma lista não ordenada.
  12. li: Define cada item dentro de uma lista ordenada ou não ordenada.
  13. hr: Cria uma linha reta atuando com separador entre partes da página
  14. br: Faz pular uma linha. Usado para adicionar linhas de espaços.

Outras tags não tão básicas mas que valem a lembrança são: del: Usada para marcar texto deletado.
Exemplo: Minha cor favorita é verde. azul escuro.

font: Esta tag foi usada no HTML4. Não é suportada no HTML5.
Era usada para definir font size, cor do texto e font-face.
Seu uso foi substituído pelo CSS.

Entendendo a diferença entre listas ordenadas e não ordenadas

Listas ordenadas são itens mostrados em forma de lista, sendo eles numerados.
Já as listas não ordenadas são mostradas como "tópicos", iniciando com uma marcação de um ícone como um círculo. Através do CSS podemos alterar os símbolos ou números que precedem cada item da lista.

Lista Ordenada:

  1. Banana
  2. Maçã
  3. Laranja

Lista Não Ordenada:
  1. Brócolis
  2. Cebola
  3. Tomate
